## Структура компонентов

### Глобальные компоненты

| Компонент | Страницы | Описание |
|---|---|---|
| Header | Все | Навигация, поиск, иконки корзины/wishlist |
| GoldTicker | Все | Актуальная цена XAU в шапке |
| Footer | Все | Ссылки, контакты, виджет котировок |
| CookieBanner | Первый визит | GDPR/уведомление о cookies |
| Breadcrumbs | Каталог, Товар, Блог | Хлебные крошки |
| SearchModal | При активации поиска | Модальный поиск |

### Компоненты страниц

**Главная (/)**
- HeroSlider — баннер с акцентными товарами
- GoldPriceBanner — текущие котировки крупно
- FeaturedProducts — 4-8 популярных товаров
- HowItWorks — 3 шага покупки
- WhyUs — преимущества компании
- LatestBlogPosts — 3 последние статьи
- ContactCTA — призыв к действию / форма

**Каталог (/catalog)**
- FilterSidebar — фильтры (тип, вес, цена)
- SortBar — сортировка и переключение вида
- ProductGrid / ProductList — сетка или список товаров
- ProductCard — карточка товара
- Pagination — постраничная навигация

**Страница товара (/catalog/[slug])**
- ProductGallery — галерея изображений
- ProductInfo — название, цена, характеристики
- AddToCart — кнопка покупки с выбором количества
- PriceCalculator — калькулятор (вес → цена)
- ProductTabs — описание / характеристики / документы
- RelatedProducts — похожие товары

**Котировки (/prices)**
- PriceChart — интерактивный график
- PriceTable — таблица цен по весу
- GoldCalculator — калькулятор инвестиций

**Корзина (/cart)**
- CartItems — список товаров
- CartSummary — итоговая сумма
- PriceChangeAlert — уведомление об изменении цены

| Средний | /cities/[city] | SSG |
